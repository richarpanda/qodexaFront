# Qodexa - Landing Page

Landing page profesional para Qodexa, una empresa de servicios tecnologicos que ofrece desarrollo de software, automatizacion y soluciones digitales.

## Servicios

- **Paginas Web** - Landing pages, sitios profesionales y e-commerce
- **Chatbots con IA** - Asistentes virtuales y atencion automatizada
- **Automatizacion** - Integracion de herramientas y flujos automatizados
- **Apps a la Medida** - Desarrollo de aplicaciones web y moviles
- **Tech Partner** - Servicio de equipo tecnologico bajo demanda

## Stack Tecnologico

- **Framework**: Next.js 15 (App Router)
- **Estilos**: Tailwind CSS v4
- **Componentes**: shadcn/ui
- **Animaciones**: Framer Motion
- **Fuentes**: Geist Sans y Geist Mono
- **Iconos**: Lucide React

## Estructura del Proyecto

```
app/
├── page.tsx              # Pagina principal
├── paginas-web/          # Servicio de paginas web
├── chatbots/             # Servicio de chatbots IA
├── automatizacion/       # Servicio de automatizacion
├── apps/                 # Servicio de desarrollo de apps
├── tech-partner/         # Servicio Tech Partner
├── layout.tsx            # Layout principal
└── globals.css           # Estilos globales

components/
├── navbar.tsx            # Navegacion principal
├── footer.tsx            # Pie de pagina
├── hero-section.tsx      # Hero con imagen de fondo
├── service-card.tsx      # Tarjetas de servicios
├── animated-section.tsx  # Componentes con animaciones
├── cta-section.tsx       # Secciones de llamada a la accion
├── testimonial-section.tsx # Testimonios de clientes
├── promo-banner.tsx      # Banner de promociones
├── trust-badges.tsx      # Badges de confianza
├── urgency-elements.tsx  # Elementos de urgencia (precios, contadores)
├── faq-section.tsx       # Preguntas frecuentes
├── whatsapp-button.tsx   # Boton flotante de WhatsApp
└── section-divider.tsx   # Separadores de secciones
```

## Instalacion

```bash
# Clonar el repositorio
git clone <repo-url>

# Instalar dependencias
pnpm install

# Iniciar servidor de desarrollo
pnpm dev
```

## Despliegue

El proyecto esta configurado para desplegarse en Vercel:

```bash
# Desplegar a produccion
vercel --prod
```
